# AI Юрист - Система создания и анализа юридических документов

Полнофункциональная система для автоматизированного создания договоров и анализа юридических документов с использованием AI-технологий для локального развертывания на XAMPP.

## 🚀 Основные возможности

### ✅ Готовые функции:
- **Система регистрации/авторизации** с современными защитами
- **Генератор договоров** из готовых шаблонов
- **Анализатор документов** с выявлением рисков
- **Управление документами** пользователя
- **Экспорт документов** в различных форматах
- **Безопасность**: хеширование паролей Argon2ID, CSRF защита, rate limiting

### 📋 Шаблоны договоров:
- Договор аренды помещения
- Договор оказания услуг
- Договор поставки (расширяется)
- Трудовые договоры (расширяется)
- NDA соглашения (расширяется)

### 🔍 Анализ документов:
- Проверка обязательных реквизитов (ИНН, ОГРН)
- Выявление юридических рисков
- Проверка соответствия ГК РФ и ТК РФ
- Анализ ясности формулировок
- Рекомендации по улучшению

## 🛠 Установка

### Требования:
- XAMPP (Apache + MySQL + PHP 7.4+)
- Браузер с поддержкой JavaScript

### Пошаговая установка:

1. **Скопируйте файлы** в папку `htdocs/ai_lawyer/`

2. **Запустите XAMPP** и убедитесь, что Apache и MySQL работают

3. **Откройте браузер** и перейдите по адресу:
   ```
   http://localhost/ai_lawyer/setup.php
   ```

4. **Нажмите "Установить базу данных"** - система автоматически создаст:
   - База данных `ai_lawyer_db`
   - Все необходимые таблицы
   - Тестовые шаблоны договоров

5. **Готово!** Перейдите на главную страницу:
   ```
   http://localhost/ai_lawyer/
   ```

## 🔒 Безопасность

Система включает современные методы защиты:

- **Аутентификация**: Хеширование паролей Argon2ID
- **Сессии**: Безопасные настройки cookies, регенерация ID
- **CSRF защита**: Токены для всех форм
- **Rate limiting**: Защита от брутфорса (5 попыток за 15 минут)
- **XSS защита**: Экранирование всего пользовательского ввода
- **Валидация**: Строгая проверка всех входных данных
- **Аудит**: Логирование всех действий пользователей

## 🎯 Использование

### Регистрация и вход:
1. Создайте аккаунт на `/auth/register.php`
2. Войдите в систему на `/auth/login.php`

### Создание договоров:
1. Перейдите в "Генератор договоров"
2. Выберите тип договора (аренда, услуги и т.д.)
3. Заполните форму с данными сторон
4. Получите готовый договор с AI-улучшениями

### Анализ документов:
1. Перейдите в "Анализ документов"
2. Загрузите файл или вставьте текст
3. Получите детальный отчет с рисками и рекомендациями

## 🧠 AI-функционал

Система использует встроенные алгоритмы анализа без внешних API:

- **Анализ структуры**: Проверка наличия обязательных разделов
- **Юридическая проверка**: Соответствие российскому законодательству
- **Оценка рисков**: Выявление потенциальных проблем
- **Улучшение текста**: Стандартизация формулировок
- **Полнота документа**: Оценка от 0 до 100 баллов

Автоматически создаются таблицы:

## 🐛 Устранение неполадок

### Ошибка подключения к БД:
- Убедитесь, что MySQL запущен в XAMPP
- Проверьте права пользователя root
- Проверьте порт MySQL (обычно 3306)

### Ошибка прав доступа:
- Убедитесь, что папка `temp/` доступна для записи
- На Windows: правый клик → Свойства → Безопасность

### Проблемы с сессиями:
- Очистите cookies браузера
- Проверьте настройки PHP session
