# 🤖 ИИ-интеграция для AI Юрист

## Обзор

Система AI Юрист теперь оснащена мощной интеграцией с **OpenAI GPT-4o** для проведения глубокого юридического анализа документов в соответствии с российским законодательством.

## 🚀 Новые возможности

### Интеллектуальный анализ документов
- **Глубокий анализ рисков** - выявление потенциальных юридических проблем
- **Соответствие российскому праву** - проверка на нарушения ГК РФ, ТК РФ и другого законодательства
- **Умные рекомендации** - конкретные предложения по улучшению документов
- **Критические проблемы** - выделение наиболее важных вопросов
- **Оценка полноты** - анализ недостающих разделов и условий

### Поддерживаемые форматы
- **DOCX** - автоматическое извлечение текста
- **DOCX** - полная поддержка Microsoft Word
- **TXT** - текстовые файлы

### Типы документов
- Договоры/Контракты
- Трудовые договоры
- Договоры аренды
- Договоры оказания услуг
- Договоры купли-продажи
- Прочие юридические документы

## 🛠 Использование

### 1. Основной интерфейс
Перейдите в раздел **"Анализ документов"** в панели управления:
- Выберите тип документа
- Загрузите файл (DOCX, TXT)
- Отметьте "Использовать ИИ-анализ"
- Нажмите "Проанализировать с ИИ"

### 2. Программное использование
```php
// Создание анализатора с ИИ
$analyzer = new DocumentAnalyzer(true);

// Анализ документа
$result = $analyzer->analyzeDocument(
    $documentText, 
    $userId, 
    $filename, 
    $documentType
);

if ($result['success']) {
    $analysis = $result['analysis'];
    // Обработка результатов
}
```

### 3. Прямое использование OpenAI класса
```php
$openAI = new OpenAIAnalyzer();
$result = $openAI->analyzeDocument($text, 'contract');
```

## ⚙️ Конфигурация

API-ключ OpenAI уже настроен в `config/security.php`:
```php
define('OPENAI_API_KEY', 'ваш_ключ_здесь');
```

### Параметры анализа
- **Модель**: GPT-4o
- **Temperature**: 0.3 (для стабильных результатов)
- **Max tokens**: 4000
- **Формат ответа**: JSON

## 🎯 Особенности для российского рынка

### Учитываемое законодательство:
- **Гражданский кодекс РФ** - договорные отношения
- **Трудовой кодекс РФ** - трудовые договоры
- **Налоговый кодекс РФ** - налоговые аспекты
- **Валютное законодательство** - валютные операции
- **Антимонопольное законодательство** - конкуренция
- **Защита прав потребителей** - потребительские договоры

### Специализированные проверки:
- Обязательные реквизиты (ИНН, ОГРН)
- Соответствие валютному законодательству
- Правильность указания НДС
- Проверка сроков и ответственности
- Анализ форс-мажорных обстоятельств

## 🔄 Fallback механизм

Система автоматически переключается на базовый анализ, если:
- API недоступен
- Превышен лимит запросов
- Произошла ошибка при обращении к ИИ

## 📊 Интерфейс
